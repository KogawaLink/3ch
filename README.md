### サイト説明
掲示板サイトに運営側からの通知を加えたサイトです。<br>
狙っているユーザー層は後で追記しますが、その層を獲得するために既存の掲示板サイトに通知機能を加えました。<br>
<br>
<br>
### このアプリでできること
* スレッド作成
* スレッド削除
* コメント書き込み
* 画像投稿
* スレッド検索

※こちら側からメンテナンス情報や、デザイン、追加機能を変更や搭載した際ユーザーに通知を送ることができます。
